整理下windows下常用的安装的软件

必备神器

  • Everything 根据文件名搜索文件(夹)
  • Wox
  • FileLocator 可以搜索文件内容
  • IDM 下载利器
  • Tim
  • Wechat

高效办公

  • MathType Office里输入公式
  • islide tools 做PPT高效率工具
  • winrar 或者 Bandizip
  • Adobe Acrobat / Adobe Reader
  • Origin 作图软件
  • sticky notes 便利贴，系统内置
  • 万彩办公大师
  • 百度云
  • 度盘下载器
  • FileGee文件同步
  • BtSync 分享文件
  • Syncthing 同步文件
  • 网易邮箱大师

影音娱乐

  • Chrome 浏览器及其众多的第三方开发版，如Cent, Brave
  • Potplayer 影音播放器
  • 网易云音乐

系统工具

  • cmder 替代 cmd
  • notepad++ 替代 notepad
  • Dism++
  • Uninstall tools 软件卸载清理
  • CCleaner Pro 文件清理
  • sysinternals-suite 系统工具集
  • NirSoft windows管理软件集合
  • Complete Internet Repair 网络问题修复

远程桌面

  • Xshell
  • Teamviewer 远程连接
  • Anydesk
  • 向日葵

编程

  • msys2
  • Python
  • textlive Latex包
  • Texmacs 所见即所得
  • neovim vim的替代版本
  • MATLAB
  • Mathematica
  • Chocolatey windows下的软件管理工具，不好

科研

  • Endnote 看文献，整理文献
  • HistCitePro 找出文献之间的关系
  • Scihub 临时下载文献
  • Material Studio 搭结构
  • VESTA 看结构
  • p4vasp 数据后处理包
  • vaspkit 数据后处理包

FQ

  • SSTap
  • SSR
  • V2Ray
